How does C-CHAIN optimize the transaction duration of the remote check clearing system?

C-Chain platform

In order to improve the check clearing system, an interface and an inter-bank blockchain network allows a bank teller A to initiate a check transaction from a bank B in order to optimize the duration of the real-time transaction and avoid any kind of error with more security and transparency between banks. The C-Chain platform is an inter-bank Blockchain network, it aims to provide a high quality real-time remote compensation system.
